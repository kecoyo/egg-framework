# @kecoyo/egg-framework

Awesome Egg.js Web Framework and Plugin.

## QuickStart

```bash
$ npm install @kecoyo/egg-framework
```

publish your framework to npm, then change app's framework config:

```js
// {app_root}/index.js
{
  "name": "egg-app",
  "egg": {
    "framework": "@kecoyo/egg-framework"
  }
}
```
